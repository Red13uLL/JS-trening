JavaScript Numbers

You can use the global JavaScript function isNaN() to find out if a value is not a number:

Watch out for NaN. If you use NaN in a mathematical operation, the result will also be NaN:

Example:
let x = NaN;
let y = 5;
x + y;

If you use NaN in a mathematical operation, the result can be a concatenation(повезан у ланац):

Example
let x = NaN;
let y = "5";
x + y;

The typeof NaN is number:

Example:
let x = NaN;
typeof x;
